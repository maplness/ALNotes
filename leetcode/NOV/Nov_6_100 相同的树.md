# Nov_6_100 相同的树
![](https://tva1.sinaimg.cn/large/0081Kckwgy1gkfvskpd9rj30vu0u0juw.jpg)

中序遍历过程中判断有不合适的直接return false
``` C++
class Solution {
public:
    bool isSameTree(TreeNode* p, TreeNode* q) {
        if(!p && !q) {
            return true;
        }else if(!p || !q){
            return false;
        }else if(p->val != q->val){
            return false;
        }else{
            return isSameTree(p->left,q->left) && isSameTree(p->right , q->right);
        }
        
    }
};
```
